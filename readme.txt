A plugin that integrates TextSanity with WordPress.

== Description ==

The TextSanity plugin is a plugin that allows WordPress users to connect to their TextSanity account. The plugin creates an admin section where you can add, modify, and control forms where visitors to your website can add their phone number.

== Installation ==

1. Upload textsanity to the `/wp-content/plugins` directory.
2. Login to the Wordpress admin panel and go to the the plugins page. On the plugins page, activate the TextSanity plugin.
3. Navigate to the TextSanity > Account Connection page.
4. Click on the Connect TextSanity Account and follow the steps to authorize WordPress to be able to connect with your TextSanity account.
5. The TextSanity plugin should now be fully active.
6. Navigate to the TextSanity > Texting Widgets page to start adding forms to your site.
